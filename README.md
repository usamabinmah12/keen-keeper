Project Name

  Keen-Keeper

Short Description

  Keen-Keeper is a friend list management web application where users can store and manage their contacts. It allows viewing detailed              information and interacting with friends through call, text, and video features, along with visual insights using charts.

Technologies Used
    React
    HTML
    Tailwind CSS
    React Router
3 Key Features
     Friend List Management – Add, view, and manage friend details easily
     Communication Features – Options for call, text, and video interaction
     Data Visualization – Pie chart representation of friend-related data
